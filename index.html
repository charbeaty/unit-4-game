Collect the Seashells

Rules of the game:

    • The rules of the game are pretty simple: Collect the seashells, and try to match the random number!
      To collect the seashells, grab your mouse (preferably a computer mouse), and start clicking.
      Each seashell has a value assigned to it, but don't try to memorize that value, because it changes on each turn! Muahahaha!
      Once a seashell is clicked, it's value will be added to your total score.
      To win, match the value of the seashell to the random number. If the value of the seashells goes above the random number, you have lost.
      You will have the chance to redeem yourself, because once you lose (or win) the game will start all over.
