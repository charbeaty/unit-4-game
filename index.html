Collect the Crystals

The rules of the game are pretty simple: Collect the crystals, and try to match the random number! Each crystal has a value assigned to it, but don't try to memorize that value, because it changes on each turn! Once a crystal is clicked, it's value will be added to your total score. To win, match the value of the crystal to the random number. If the value of the crystals goes above the random number, you have lost.
