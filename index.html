Collect the Crystals

Rules of the game:

    • The rules of the game are pretty simple: collect the crystals!
      To collect the crystals, grab your mouse and start clicking. Each crystal has a value assigned to it, but don't try to memorize that value, because it changes on each turn!
      Once a crystal is clicked, it's value will be revealed and the value will be added to your total score.
      To win, match the value of the crystals to the random number. If the value of the crystals goes above the random number, you have lost.
      After each win or loss, the game will restart with a new random number and the value of each crystal will change, and the fun starts over!
